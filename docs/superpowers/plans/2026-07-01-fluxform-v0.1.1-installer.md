# FluxForm v0.1.1 Installer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship FluxForm v0.1.1 with a standard Inno Setup installer for the WPF desktop app.

**Architecture:** Keep the existing WPF publish output as the installer input, then add a focused Inno Setup layer that packages only the desktop app and bundled FFmpeg files. Keep installer generation optional in release checks so machines without Inno Setup can still build and test the project.

**Tech Stack:** .NET 9, WPF, PowerShell publish scripts, Inno Setup 6, xUnit string-based repository tests.

---

## File Structure

- `Directory.Build.props`: update shared app version metadata from `0.1.0` to `0.1.1`.
- `installer/FluxForm.iss`: new Inno Setup script for the WPF installer.
- `scripts/publish-installer.ps1`: new PowerShell entry point that publishes WPF with FFmpeg and invokes `ISCC.exe`.
- `scripts/release-check.ps1`: add optional `-BuildInstaller` switch and call the installer script only when requested.
- `README.md`: update normal user instructions to prefer the installer and document SmartScreen expectations.
- `AGENTS.md`: document the new installer publish command and v0.1.1 distribution model.
- `FluxForm.Tests/DevelopmentWorkflowTests.cs`: update version assertions and release workflow expectations.
- `FluxForm.Tests/PublishConfigurationTests.cs`: add tests for installer script/configuration and release-check optional installer behavior.

- [ ] **Step 3: Create the installer publish script**

Create `scripts/publish-installer.ps1` with:

```powershell
param(
    [ValidateSet('Debug', 'Release')]
    [string]$Configuration = 'Release',
    [string]$Runtime = 'win-x64',
    [string]$InnoSetupCompilerPath
)

$ErrorActionPreference = 'Stop'
Set-Location (Split-Path $PSScriptRoot -Parent)

& .\scripts\publish-wpf.ps1 -Configuration $Configuration -Runtime $Runtime -BundleFFmpeg
if ($LASTEXITCODE -ne 0) { exit $LASTEXITCODE }

$candidates = @()
if (-not [string]::IsNullOrWhiteSpace($InnoSetupCompilerPath)) {
    $candidates += $InnoSetupCompilerPath
}

$candidates += @(
    "${env:ProgramFiles(x86)}\Inno Setup 6\ISCC.exe",
    "$env:ProgramFiles\Inno Setup 6\ISCC.exe",
    "${env:LOCALAPPDATA}\Programs\Inno Setup 6\ISCC.exe"
)

$compiler = $candidates | Where-Object {
    -not [string]::IsNullOrWhiteSpace($_) -and (Test-Path -LiteralPath $_)
} | Select-Object -First 1

if (-not $compiler) {
    throw 'Inno Setup compiler was not found. Install Inno Setup 6 or pass -InnoSetupCompilerPath with the full path to ISCC.exe.'
}

& $compiler ".\installer\FluxForm.iss"
if ($LASTEXITCODE -ne 0) { exit $LASTEXITCODE }
```
